# 🛠️ Required Tools

> Having the right tools prevents most assembly headaches. This list is drawn from community experience — no gadgets, just the essentials.

## ⚡ Wiring tools

| Tool | Budget | Serious | Notes |
|------|--------|---------|-------|
| **Wire stripper** | AliExpress | Engineer PA-05 | Essential for terminals |
| **JST connector kit** | AliExpress full set | — | Order extra, you'll lose some |
| **JST PH/XH crimper** | IWISS SN-01BM | Engineer PA-09 | PA-09 handles all common formats |
| **Multimeter** | UNI-T UT61E | Fluke 17B+ | For continuity and voltage checks |
| **Soldering iron** | TS80P | Pinecil V2 | — |

> **Pro tip:** invest in the crimping tool. A bad crimp creates intermittent problems that are impossible to diagnose. The **Engineer PA-09** is the Voron community consensus.

| Tool | Notes |
|------|-------|

| **300×300mm mirror** | Temporary print surface for first tests |
| **IPA 90%+** | Bed cleaning before printing |
| **Nitrile gloves** | Avoid fingerprints on print surface |

## ✅ Optional (but useful)

| Tool | Use |
|------|-----|
| **USB oscilloscope (ex: DSO150)** | Advanced electronics diagnostics |
| **Clamp ammeter** | Checking axis current draw |
| **Cordless drill / driver** | Faster frame assembly |
| **Magnifying lamp** | Precision crimping and soldering |

---

> 💡 **Keep in mind:** for Voron, crimping Molex/JST connectors is the most time-consuming operation and the biggest source of errors. Buy 30% more terminals than you think you need and test before final assembly.
